  1. Problem: Write routines that will compute and return the $l_1$, $l_2$, and $l_\infty$ norms (lengths) of a given vector of arbitrary length. The routines should use as input, a vector and the length of the input vector and output a double value that is the length of the vector. Add the three routines to your software library. Add an entry in your software manual for each of the routines. Make sure that you include simple examples illustrating the usage of the functions/methods you have corrected.
  2. Problem: Write routines that will compute the absolute and relative errors in the approximation of an "exact" vector and a vector used to approximate the exact vector. The routines should take as input two vectors, $x$ and $y$, and the length of the two vectors. The output should be a single double value that represents the error. Note that there are six total routines that need to be developed. There are three measures of error for both the absolute error and three for the relative error. The three routines should compute the errors in the $l_1$, $l_2$, and $l_\infty$ norms as defined in the previous problem. As usual, add entries to your software manual with simple examples.
  3. Problem: Write separate routines for each of the following vector operations
    1. vector addition
    2. vector subtraction
    3. multiplying a scalar into a vector
    4. the inner product of two vectors of the same length
    5. the cross product of two vectors of the same length
    Add entries to your software manual to cover these routines. As in all homework problems, include simple examples of how your code works.
  4. Problem: Write separate routines to compute the Frobenius, $l_1$, and $l_\infty$ norms of a matrix. Add entries to your software manual for these routines.
  5. Problem: Write separate routines for each of the following matrix operations.
    1. matrix addition
    2. matrix subtraction
    3. the transpose a matrix
    4. trace of a matrix
    5. multiplying a scalar into a matrix
    6. multiplication of a matrix into a vector
    7. multiplication of two matrices
    Add entries to your software manual for each of the routines you write.
  6. Problem: Write routines for the following additional matrix operations.
    1. the outer product of two matrices
    2. the Kronecker product of two matrices
    3. determinant of a matrix
    Add entries to your software manual for each of the routines you write.
  7. Problem: Write routines that take advantage of parallel/concurrent computing to perform the following matrix operations.
    1. the product of a matrix and vector
    2. the product of two matrices
    You do not need to add an entry to the software manual in this case. However, you need to give examples to verify your code works.
  8. Problem: Write routines that implement the following vector operations.
    1. the outer product of two matrices,
    2. the inner product or dot product of two vectors, and
    3. the cross product of two vector.
  9. Problem: In a standard linear algebra course the process of changing bases is presented. For this problem, define an algorithm for determining an othogonal basis from a general basis in two dimensions. This means taking a pair of vectors, ${\bf v_1}$ and ${\bf v_2}$ and determining a pair of unit vectors that form an orthogonal pair of vectors, ${\bf u_1}$ and ${\bf u_2}$. Include an entry in your software manual with some simple examples.
