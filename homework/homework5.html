Math 4610 Fundamentals of Computational Mathematics: Homework 5 Problems

Due: Monday, October 8, 2018

  1. Problem: Write code that computes the induced matrix norm using the $l_1$ and $l_\infty$ norms. Include entries in your software manual along with some simple examples to illustrate the usage of the code that you have created.
  2. Problem: Write piece of code that will initialize a square matrix so that the matrix has uniformly distributed numbers and is symmetric and diagonally dominant. The code should (1) initialize the entries in the upper triangular and diagonal entries of the matrix, (2) reflect the values in the upper triangular part of the matrix into the lower triangular portion of the matrix, and (3) add a large enough entry (in magnitude) to the diagonal entry to ensure the matrix is diagonally dominant. Also, write a piece of code that will generate the right hand side of a system of equations by multiplying the matrix above into a vector of ones. You should put an entry in your software manual that documents this code and how to use the code to generate test problems.
  3. Problem: Write code that implements Jacobi iteration for the solution of linear systems of equations. Test your code on a random, symmetric, diagonally dominant matrix and right hand side as described in the previous problem. Make sure that you include an entry in your software manual.
  4. Problem: Write code that implements Gauss-Seidel iteration for the of linear systems of equations. Test your code on a random, symmetric, diagonally dominant matrix and right hand side as described in the previous problem. Make sure that you include an entry in your software manual.
  5. Problem: Write code that implements the method of Steepest Descent (the Gradient method) for the solution of linear systems of equations. Test your code on a random, symmetric, diagonally dominant matrix and right hand side as described in the previous problem. Make sure that you include an entry in your software manual.
  6. Problem: Write code that implements the conjugate gradient method for the solution of linear systems of equations. Test your code on a random, symmetric, diagonally dominant matrix and right hand side as described in the previous problem. Make sure that you include an entry in your software manual.
  7. Problem: In this problem, you will need to implement a matrix-vector code that uses OpenMP. Use the code to perform the matrix-vector operation in your Jacobi iteration code. Create a new version of the Jacobi iteration that uses the OpenMP implementation of the produce. Test your code on the same matrix that you used for the test of the vanilla Jacobi iteration.
